# 로또

## 기능 목록

### 입력
- 로또 구입 금액 입력 받기
- 구입 금액 1,000원 단위 검증
- 구입 금액 양수 검증
- 당첨 번호 입력 받기 (쉼표로 구분)
- 당첨 번호 6개 검증
- 당첨 번호 1~45 범위 검증
- 당첨 번호 중복 검증
- 보너스 번호 입력 받기
- 보너스 번호 1~45 범위 검증
- 보너스 번호와 당첨 번호 중복 검증

### 로또 발행
- 구입 금액으로 로또 구매 개수 계산
- 1~45 범위에서 중복되지 않는 6개 번호 생성
- 로또 번호 오름차순 정렬
- 발행한 로또 개수 및 번호 출력

### 당첨 확인
- 로또 번호와 당첨 번호 일치 개수 계산
- 보너스 번호 일치 여부 확인
- 당첨 등수 판정 (1등~5등)
- 등수별 당첨 개수 집계

### 수익률 계산
- 총 당첨 금액 계산
- 수익률 계산 (당첨 금액 / 구입 금액 * 100)
- 소수점 둘째 자리에서 반올림

### 출력
- 구매한 로또 개수 및 번호 출력
- 당첨 통계 출력
- 총 수익률 출력

### 예외 처리
- 잘못된 입력 시 "[ERROR]" 메시지 출력
- 해당 부분부터 재입력 받기

## 커밋 컨벤션

### 커밋 메시지 구조
```
<type>: <subject>
```

### Type
- `feat`: 새로운 기능 추가
- `fix`: 버그 수정
- `docs`: 문서 수정
- `refactor`: 코드 리팩토링
- `test`: 테스트 코드 추가 또는 수정
- `chore`: 빌드 업무 수정

### 작성 규칙
- 제목은 50자 이내
- 제목 첫 글자는 대문자
- 제목 끝에 마침표 금지
- 제목은 명령문으로 작성
